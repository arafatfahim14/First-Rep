How to Start!

    In GitHub

  • Go to create new repository
  • Set Owner
  • Give a Repository Name (Example: JV)
  • Select Public or Private
  • Create Repository
  • Copy Important Information

    In Desktop

  • Create a folder as Project at C Drive (Must be same as Repository Name)
  • Open JV in VS Code
  • Open terminal (cmd)
  • git --version
  • cd \Projects
  • cd JV
  • git init
  • git add .
  • git commit -m"first commit"
  • git branch -M main
  • Set the origin: (Example: git remote add origin https://github.com/arafatfahim14/JV.git)
  • git push -u origin main
  • After Every Edit
      git add .
      git commit -m"xyz" (Here XYZ is a meaningful commit message)
      git push
  • Create a domain
      Go to repository settings
      Go to pages
      Select main as Branch, save it and refresh
      If domain is not created: create a domain (Example: https://arafatfahim14.github.io/JV/)
  • git remove -v (It will show the file location)
