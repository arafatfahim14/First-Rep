How to Start!

    In GitHub

  • Go to create new repository
  • Set Owner
  • Give a Repository Name (Example: JV)
  • Select Public or Private
  • Create Repository
  • Copy Important Information

    In Desktop

  • Create a folder as Project at C Drive (Must be same as Repository Name)
  • Open JV in VS Code
  • Open terminal (cmd)
  • git --version
  • cd \Projects
  • cd JV
  • git init
  • git add .
  • git commit -m"first commit"
  • git branch -M main
  • Set the origin: (Example: git remote add origin https://github.com/arafatfahim14/JV.git)
  • git push -u origin main
  • After Every Edit
      git add .
      git commit -m"xyz" (Here XYZ is a meaningful commit message)
      git push
  • Create a domain
      Go to repository settings
      Go to pages
      Select main as Branch, save it and refresh
      If domain is not created: create a domain (Example: https://arafatfahim14.github.io/JV/)
  • git remove -v (It will show the file location)
  • while adding a photo we must fully copy it. (Example:To store "1.jpg" image we must write "1.jpg" not "1.JPG" or something else.)

    Advance Git

  • For official it must me Private
  • Must invite a collaborator by setting the repository if he wants to add
  • A user working as collaborator in a project can clone it
      git clone https_link_of_the_project
  • Fork - A user who is not even collaborator, but wants to contribute can clone it using fork
      IN VS terminal
      git branch
      git checkout main
      git checkout -b add-fruits (Let's assume "fruits" is a new branch.)
      Update the code as you want
      Add and commit like the same way
      If you push like the same way you will get a error. It will say, what you have to write. However, this might looks like -"--set-upstream origin add-fruits" (fruits is the current branch)
      To request the owner to add it in main branch, he has to sent pull request by GitHub
      Finally, owner can review or merge pull request. And if owner wants to pull the updated one in his vs code, he need to write "git pull"
